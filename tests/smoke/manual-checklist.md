# Manual Smoke Checklist

- `npm run dev` starts the Vite app.
- Register, log in, and log out locally.
- Launch the 3D trainer, confirm pointer lock, aiming, shooting, pause, resume, replay, and menu return.
- Open Leaderboard, Customize, Shop, Progress, Social, Reaction Test, Profile, and Updates.
- Equip cosmetics in Customize, buy cosmetics in Shop, and confirm persistence after reload.
- Trigger a reaction test run and reset it.
- Open Profile edit mode, update avatar data, and confirm the crosshair preview still renders.
